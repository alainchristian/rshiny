
# Youth Survey Dashboard Styling Instructions

Claude Code, please follow these instructions to enhance the styling of the Youth Survey dashboard:

## KPI Cards (`ys-kpi-row`)
- Adjust the padding and spacing between the KPI cards to create a balanced layout.
- Use consistent font sizes and weights for the KPI labels, values, and subtitles.
- Consider adding icons or visuals to enhance the visual appeal of the KPI cards.
- Ensure proper color contrast for readability.

## Charts (`ys-chart`)
- Customize the chart colors to match the ASYV branding or create a visually appealing color scheme.
- Add chart titles and labels to provide clear context for the data being displayed.
- Use appropriate font sizes and styles for chart labels and legends.
- Adjust the chart margins and spacing to ensure proper fit within the card.
- Implement hover interactions or tooltips to display additional information when hovering over chart elements.

## Tables (`ys-table`)
- Style the table headers with a distinct background color and font style to differentiate them from the table data.
- Adjust the column widths to ensure proper readability and prevent content overflow.
- Use zebra striping or alternating row colors to enhance visual separation between rows.
- Implement pagination or infinite scrolling for tables with a large number of rows.

## Filters (`ys-filters`)
- Organize the filters in a logical and intuitive manner, grouping related filters together.
- Use consistent styling for filter labels, dropdowns, and checkboxes.
- Provide clear instructions or placeholders to guide users in selecting appropriate filter values.
- Adjust the spacing and alignment of filter elements to create a clean and organized layout.

## Typography
- Choose a readable and visually appealing font family for the dashboard.
- Use appropriate font sizes and weights for different levels of text hierarchy (e.g., headings, labels, values).
- Ensure proper line spacing and paragraph margins for optimal readability.
- Apply consistent text alignment and spacing throughout the dashboard.

## Color Scheme
- Select a color palette that aligns with the ASYV branding and creates a cohesive visual experience.
- Use colors consistently throughout the dashboard for different elements (e.g., headers, buttons, charts).
- Ensure sufficient color contrast between text and background colors for accessibility.

## Responsive Design
- Test the dashboard on various screen sizes and devices to ensure proper responsiveness.
- Use CSS media queries to adjust the layout and styling based on different screen sizes.
- Optimize the dashboard for both desktop and mobile devices, considering touch interactions and smaller screen resolutions.

## Consistency and Branding
- Maintain consistency in styling across different sections and elements of the dashboard.
- Incorporate ASYV branding elements, such as logos, colors, and imagery, to create a cohesive visual identity.
- Ensure that the overall design aligns with the ASYV brand guidelines and style.

Remember to test the styling changes thoroughly to ensure a seamless and visually appealing user experience. If you encounter any specific challenges or have further questions, don't hesitate to reach out for assistance.

Happy styling!
